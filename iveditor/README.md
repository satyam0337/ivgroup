# IVEditor

## How to use locally

1. Make sure you have NodeJS installed, if not install it first.

2. Then from your workspace run the following commands. This will start a development server for IVEditor.

    ```sh
    $ cd iveditor
    $ npm install
    $ npm run dev
    ```

3. Run Tomcat Server from Ecllipse
4. Log into "ivcargo" account
5. Open this URL: http://localhost:8082/ivcargo/iveditor.do?pageId=340&eventId=12&modulename=iveditor
